# Privacy Policy — Cat Mode — Keyboard Guard

_Last updated: 6 June 2026_

Cat Mode — Keyboard Guard ("the extension") is designed with privacy as a
first principle.

## What we collect
Nothing. The extension does not collect, store, or transmit any personal or
sensitive user data. It does not track your browsing, read page content, or
log keystrokes for any external purpose.

## What is stored, and where
The extension saves only its own settings on your device using Chrome's
`chrome.storage` API:

- whether Cat Mode is currently on or off, and
- the keyboard shortcut you choose to unlock it.

This information never leaves your browser. It is not sent to us or to any
third party, because the extension has no servers and makes no network
requests.

## Permissions
- **storage** — used only to remember the two settings listed above.
- **Host access (all sites)** — required so the extension can block stray
  keyboard, mouse, and trackpad input on whatever page you are viewing while
  Cat Mode is enabled. It only blocks input events; it does not read or
  transmit page content.

## Data sharing
We do not sell, share, or transfer any user data to third parties, because no
user data is ever collected.

If this policy changes, the updated version will be posted at this same URL.
