Cat Mode

Block your keyboard & trackpad so a pet walking across them can't pause your video, skip ahead, or trigger random shortcuts. Unlock instantly with your own combo.

🐾 Add to Chrome View on GitHub

The Add-to-Chrome link goes live once the listing passes review. Until then, use the manual install below.

Why you'll want it

🐱

Pet-proof

One click freezes keyboard, mouse, trackpad, scroll, and touch on the page.

🔑

Your own unlock

Set a shortcut only a human would press (default Ctrl+Shift+K).

⚡

Toggle anywhere

Toolbar button or a browser-wide shortcut, even while input is blocked.

🔒

Private

No accounts, no tracking. Everything stays on your device.

How it works

  1. Click the 🐾 toolbar icon and flip Cat Mode on.
  2. A “🐾 Cat Mode ON” banner appears — input is now blocked.
  3. Press your unlock shortcut to return to normal. That's it.

Manual install (before store approval)

  1. Download the code and unzip it.
  2. Open chrome://extensions and turn on Developer mode.
  3. Click Load unpacked and select the cat-mode folder.
  4. Pin it, open a video, and toggle Cat Mode on.
